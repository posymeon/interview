
# User Management Api
## Setup

Create a .env file, in the root folder and add the following environment variables

```
PORT=4002
```
---

## Run

1. ```npm install```

2. ```npm run start```

## Build
### Build for production 

1. ```npm run clean```

2. ```npm run build```

### Build docker image 

``` docker build -t <image_name> . ```
